
# ESS 2022

This is the presentation given at the NNSP-SwedNess Neutron School 2022, held at the ESS - Lund.

### Part 1

Introduction to Linear spin-wave theory, SpinW and basic examples.

### Part 2

In depth Kagome lattice example.
